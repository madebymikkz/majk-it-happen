Description
-----------
Majk It Happen is a program for making digital to-do lists.
Tasks are added to the task list, and can then be checked off when they are done.

Features
--------
This version of Majk It Happen has the following features:

 * Handles multiple separate lists of tasks simultaneously

 * Tasks have a due date

 * Task lists can be saved to and loaded from disk
